# Dice-Roll-Game

is a simple fun two player dice game.

To Play : https://alzubi-omar.github.io/Dice-Roll-Game/

The Game Rules:

1. Player 1 starts: rolls dice --> Generate Random Number --> Display dice roll --> is it 1 ? No: Add dice to current score --> Display new score Yes: Switch Player

2. Player holds score --> Add current score to toal score --> score >= 100 ? No : Switch Player Yes : Current PLauer Wins!!

3. Player resets game --> Set all scores to 0 --> Set player 1 as starting player
